# Icon Files Guide

## Required Icons

The extension requires three icon sizes in the `images/` folder:

```
images/
├── icon-16.png   (16x16 pixels)
├── icon-48.png   (48x48 pixels)
└── icon-128.png  (128x128 pixels)
```

## Specifications

| Size | Usage | Format |
|------|-------|--------|
| 16px | Context menu | PNG with transparency |
| 48px | Extension management page | PNG with transparency |
| 128px | Chrome Web Store | PNG with transparency |

## Creating Icons

### Option 1: Use an Icon Generator
- [Favicon Generator](https://www.favicon-generator.org/)
- [Online Icon Converter](https://icoconvert.com/)
- [Icon Converter by CloudConvert](https://cloudconvert.com/icon)

### Option 2: Design Custom Icons
Recommended design for anime theme:
- **Background**: Gradient (purple to blue)
- **Icon**: Anime character silhouette or play button
- **Style**: Flat, modern, recognizable at small sizes
- **Color**: Bright accent color like cyan or pink

### Option 3: Use Placeholder Icons
For development/testing, create simple placeholder PNGs:

```bash
# Using ImageMagick
convert -size 16x16 xc:"#e94560" images/icon-16.png
convert -size 48x48 xc:"#e94560" images/icon-48.png
convert -size 128x128 xc:"#e94560" images/icon-128.png

# Or using Python with Pillow
python3 << 'EOF'
from PIL import Image

for size in [16, 48, 128]:
    img = Image.new('RGB', (size, size), color='#e94560')
    img.save(f'images/icon-{size}.png')
EOF
```

## What to Create

For this anime extension, consider:
1. A stylized anime character or mascot
2. A play button with anime aesthetics
3. Minimalist design that scales well to 16x16
4. Use the brand colors: `#e94560` (pink/red) and `#00d4ff` (cyan)

## Icon Design Tips

- **Clarity at 16px**: Keep details minimal, avoid thin lines
- **Color contrast**: Ensure icon is visible on both light and dark backgrounds
- **Consistent style**: Match the anime-themed UI design
- **Round corners**: Suggested 2-3px rounded corners
- **Transparency**: Use transparent PNG background

## Installation

1. Create the `images/` directory if it doesn't exist:
   ```bash
   mkdir images
   ```

2. Add your icon files:
   ```bash
   # Copy your icon files
   cp /path/to/icon-16.png images/
   cp /path/to/icon-48.png images/
   cp /path/to/icon-128.png images/
   ```

3. Verify files exist:
   ```bash
   ls -la images/
   ```

4. Reload extension on `chrome://extensions/`

## Troubleshooting

### Icons not showing
- Verify file paths: `images/icon-{16,48,128}.png`
- Check file format: Must be PNG
- Ensure transparency is preserved
- Reload extension after adding files

### Icon appears blurry
- Ensure image is exactly the right size (not scaled)
- Use anti-aliasing in design software
- Save with maximum quality

### Icon not showing in Chrome Web Store
- 128px icon must be exactly 128x128
- Must be PNG format
- File should be less than 1MB
- No transparency required for store (but can have it)
