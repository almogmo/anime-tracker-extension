# Chrome Web Store Listing

## Short summary
Track your anime episodes and let Netflix skip intros & jump to the next episode automatically — in any language.

## Detailed description
Anime Fans Tracker is a lightweight companion for anime watchers.

📺 Episode Tracker — Save the anime you're watching with the current episode number. The link to the page you're on is captured automatically, so you can jump right back to where you left off. Everything is stored locally on your device.

⏭️ Netflix Auto-Skip — Automatically skips the intro/recap and advances to the next episode when the end credits roll — so your binge never stops. Works in every language (it uses Netflix's built-in button identifiers, not text), and it only advances at the end of an episode, never from the start.

No accounts, no tracking, no ads. Just track and watch.
